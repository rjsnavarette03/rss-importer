Description:
This plugin automatically imports RSS feed items and creates WordPress posts from them.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Settings' â†’ 'RSS Importer' page to configure the plugin.
4. Optionally, you can manually trigger a feed import using the "Fetch Now" button.
